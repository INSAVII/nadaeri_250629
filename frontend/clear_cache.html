🔄 브라우저 캐시 완전 삭제 가이드

⚠️ 중요: updateUserBalance 오류 해결을 위한 캐시 삭제

현재 발생하는 "updateUserBalance is not a function" 오류는 브라우저 캐시 문제입니다.

1단계: 개발자 도구 열기

F12 키를 누르거나 Ctrl + Shift + I를 눌러 개발자 도구를 엽니다.

2단계: Application 탭으로 이동

개발자 도구에서 Application 탭을 클릭합니다.

3단계: Storage 영역에서 Clear storage 클릭

왼쪽 사이드바에서 Storage → Clear storage를 클릭합니다.

4단계: 강제 새로고침

Ctrl + Shift + R (또는 Cmd + Shift + R)을 눌러 강제 새로고침을 합니다.

✅ 완료 후 확인사항

1. 브라우저가 완전히 새로고침되었는지 확인

2. QName 페이지에서 "🧪 파일처리테스트" 버튼으로 테스트

3. 콘솔에서 "updateUserBalance is not a function" 오류가 사라졌는지 확인

🔧 추가 방법: 브라우저 캐시 완전 삭제

Chrome: Ctrl + Shift + Delete → "모든 시간" 선택 → "캐시된 이미지 및 파일" 체크 → "데이터 삭제"

Firefox: Ctrl + Shift + Delete → "모든 시간" 선택 → "캐시" 체크 → "지금 삭제"

📋 하이브리드 방식 서버 상태 확인

현재 실행 중인 서버들:

  • ✅ 프론트엔드: http://localhost:3003
  • ✅ 메인 API: http://localhost:8001
  • ✅ QName 서비스: http://localhost:8004
  • ✅ QText 서비스: http://localhost:8003

하이브리드 방식은 올바른 구성입니다!
